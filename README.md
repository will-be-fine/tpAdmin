# tpAdmin

### 描述
基于thinkphp6+layui开发的后台管理框架

### 说明

**运行**

进入命令行下面，执行下面指令
```
php think run -p 80
```
在浏览器中输入地址：http://localhost:80/


**路由规则**：http://127.0.0.1:8080/admin.test/hello


### 开发计划

1. 架构搭建
    * route
    * model
    * service

2. 数据库自动导入

3. 基础模块
    * 用户
    * 角色权限
    * 登录
